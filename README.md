### 2️ Python 버전 – **운영·유지보수 현실을 반영한 최종 선택**

실제 회사 및 실증 환경을 고려했을 때 다음과 같은 제약이 존재했습니다.

- C# / .NET 유지보수가 가능한 인력이 내부에 없음
- 현장 및 연구 환경에서는 Python 활용도가 훨씬 높음
- CSV 시나리오, 데이터 가공, 실험 코드 작성에 Python이 유리

이로 인해:

- **최종 운영 및 유지보수 대상은 Python 버전**
- C# 버전에서 설계한 구조와 동작 방식을
   Python(Tkinter)으로 재구현

하는 방향으로 결정되었습니다.
